SpesCoin Instructions:

1 - Synchronize with the blockchain :

    just run "Synchronize with blockchain.cmd" and wait to synchronize.

2 - Wallet

    just run "SimpleWallet.cmd" and follow the onscreen instructions and you'll be ready to go in no time!!! 

3 - Miner - this is only for local mining
 
    Before you run "Miner.cmd" please edit it and make sure you are synchronized with the blockchain.

    The command inside "Miner.cmd" shloud look similar to: miner --address 123456 --threads 6 --daemon-rpc-port 55490 --log-level 4

Pool mining intructions please visit http://pool.myspes.org

Happy mining and enjoy SpesCoin :) !!!
